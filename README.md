# Layna

ブラウザベースのプロジェクト管理／レビューツール。ShotGrid のようなワークフローを、
**OGPipeline / OGREF** の考え方を踏襲して実装しています。

- 🎬 動画・画像をアップロードし、ディレクターがチェック（OK / リテイク / NG）
- 🧩 プロジェクトの工程は **OGPipeline の工程設定を流用**（テンプレートから複製して調整可能）
- 🗂️ ショット一覧は **OGREF / OGPipeline のようにタイル表示**（サムネイル・工程ステータス付き）
- 💾 アップロードされた動画・画像は **このツール用のローカルフォルダ** (`storage/media`) に格納

## 技術構成

| レイヤ       | 使用技術                              |
| ------------ | ------------------------------------- |
| バックエンド | Node.js + Express + multer            |
| フロント     | バニラ JS（SPA）                      |
| データ保存   | ローカル JSON (`data/db.json`)        |
| メディア保存 | ローカルフォルダ (`storage/media`)    |

DB は不要です。メタデータは JSON、メディアはファイルとしてローカルに保存されます。

## セットアップ

```bash
npm install
npm start
```

ブラウザで <http://localhost:3000> を開きます。
（開発時のホットリロードは `npm run dev`。ポート変更は `PORT=8080 npm start`。）

## 使い方

1. **プロジェクト作成**：「新規プロジェクト」→ 工程テンプレート（標準CG / 2D作画 / シンプル）を選択。
   選んだテンプレートの工程がプロジェクトに複製されます（＝OGPipeline の工程設定を流用）。
2. **工程設定**：プロジェクト画面の「工程設定」から工程の追加・名称・色を編集可能。
3. **ショット追加**：`cut_010` のようなショットを作成。タイル表示されます。
4. **アップロード**：ショットタイルを開き、動画・画像をアップロード。
   バージョン（v1, v2 …）として積み上がり、サムネイルはブラウザ側で自動生成されます。
5. **ディレクターチェック**：各バージョンに対し OK / リテイク / NG / 保留を設定し、コメントを追加。
6. **工程ステータス**：ショットごとに各工程を「未着手 / 作業中 / 確認待ち / リテイク / OK」で管理。

## ディレクトリ構成

```
Layna/
├── server/
│   ├── server.js      # Express アプリ・API・アップロード処理
│   └── storage.js     # JSON 永続化レイヤ
├── public/            # フロントエンド（SPA）
│   ├── index.html
│   ├── css/style.css
│   └── js/app.js
├── data/
│   ├── seed.json      # 工程テンプレート初期値（バージョン管理対象）
│   └── db.json        # 実データ（自動生成・gitignore）
└── storage/
    ├── media/         # アップロードされた動画・画像（gitignore）
    └── thumbnails/    # 生成されたサムネイル（gitignore）
```

## 主な API

| メソッド | パス                                | 説明                             |
| -------- | ----------------------------------- | -------------------------------- |
| GET      | `/api/pipeline-templates`           | 工程テンプレート一覧             |
| POST     | `/api/projects`                     | プロジェクト作成（テンプレ流用） |
| GET      | `/api/projects/:id/shots`           | ショット一覧                     |
| POST     | `/api/projects/:id/shots`           | ショット作成                     |
| POST     | `/api/shots/:id/versions`           | 動画・画像アップロード           |
| PATCH    | `/api/versions/:id/review`          | ディレクターチェック更新         |
| PATCH    | `/api/shots/:id`                    | 工程ステータス更新               |

## データモデル（概要）

- **PipelineTemplate**：再利用可能な工程セット（OGPipeline の工程設定に相当）
- **Project**：名前・説明・工程（テンプレートから複製）
- **Shot**：プロジェクトに属し、サムネイル・工程ステータス・バージョン群を持つ
- **Version**：アップロードされた動画/画像 + レビュー（判定・コメント）
